# 模块5：Claude 用于日常工作

## 本节要点

- Claude Projects：为不同工作场景创建专属助手
- 周报自动化：从零散记录到结构化周报
- 建立个人 Prompt 库：把好用的 Prompt 存起来反复使用
- 工作流模板：常见职场场景的 Claude 用法

## Claude Projects

Projects 是 Claude 最强大的功能之一。你可以：

1. **创建专属项目**：为每个工作场景创建一个 Project
2. **上传参考资料**：把公司文档、品牌指南、历史数据传进去
3. **设置 System Prompt**：告诉 Claude 你是谁、需要什么风格的输出

### 实战：创建"周报助手"项目

**System Prompt 模板**：

> 你是我的周报助手。我在 [公司名称] 的 [部门] 工作，职位是 [职位]。
> 我每周会给你发一些零散的工作记录，请帮我：
> 1. 按项目分类整理
> 2. 用 STAR 格式（情境-任务-行动-结果）重写每条记录
> 3. 突出可量化的成果
> 4. 生成一份结构化周报

## 建立个人 Prompt 库

高效使用 Claude 的秘诀：**不要每次从零开始写 Prompt**。

### 推荐做法

1. **收集**：每次用到好的 Prompt，保存到笔记应用（Notion、备忘录等）
2. **分类**：按使用场景分类（写作、分析、代码、学习等）
3. **迭代**：不断优化 Prompt，记录什么有效、什么无效
4. **分享**：好的 Prompt 可以分享给同事，提升团队效率

### 5个必备 Prompt 模板

1. **每日总结**：把当天的工作记录整理成总结
2. **邮件回复**：根据收到的邮件生成得体的回复
3. **会议纪要**：从会议录音/笔记整理出结构化纪要
4. **数据解读**：把数据表格转化为文字洞察
5. **学习笔记**：把视频/文章内容转化为可复习的笔记

## 小练习

1. 在 Claude 中创建一个 Project，配置你的工作 System Prompt
2. 用本节的模板，生成一份本周的周报
3. 开始建立你的个人 Prompt 库，先收集3个常用模板
