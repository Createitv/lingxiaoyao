## 关于林逍遥AI

林逍遥AI 是一个面向中文用户的 AI 教育平台，专注于 AI 工具教学与 Prompt 工程实践。我们相信 AI 不应该只属于技术人员——无论你是学生、职场人、创业者还是写作者，都可以通过系统学习，把 AI 变成你最强的工作搭档。

平台提供从入门到进阶的 AI 课程、深度技术文章和实用文档，帮助你掌握 Claude、ChatGPT 等主流 AI 工具的高效使用方法。

## 课程

### Claude for Everyone — Claude AI 入门教程

面向所有人的 Claude 使用教程，零编程基础即可学习。学完这门课，你将能够把 Claude 熟练运用到日常工作和学习中。

- **认识 Claude** — 了解 Claude 是什么、能做什么、如何上手
- **学会提问** — 掌握 Prompt 的核心结构：角色 + 任务 + 背景 + 格式
- **AI 写作** — 用 Claude 撰写邮件、工作报告、文案改稿
- **AI 思考与分析** — 用 Claude 整理思路、阅读文档、辅助决策
- **AI 工作流** — 把 Claude 融入日常工作，打造个人 Prompt 库
- **进阶技巧** — 多轮对话策略、输出验证、高效人机协作
